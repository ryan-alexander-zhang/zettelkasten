## Object Relationships
### Conclusion

|                      |      |
| -------------------- | ---- |
| Solid Line           | `--` |
| Dashed Line          | `..` |
| Open/Hollow Diamond  | `o`  |
| Solid/Filled Diamond | `*`  |
| Hollow Triangle      | \|>  |
| Arrow                | `>`  |

### Implementation

```mermaid
classDiagram

class Api {
	<<interface>>
	+execute(): Void
}

class ApiImpl {
	+execute(): Void
}

Api ..|> ApiImpl: implement
```

### Extend

```mermaid
classDiagram

class Parent

class Son

Son --|> Parent: extend
```
### Aggregation

```mermaid
classDiagram

class Whole {
	-part: Part
	+Whole(part: Part)
	+setPart(part: Part)
}

class Part 

Whole o-- Part: aggregate
```

### Composition

```mermaid
classDiagram

class Whole {
	-part: Part
	+Whole()
	+getPartName(): String
}

class Part {
	-name: String
	+getName(): String
}

Whole *-- Part: Composite
```

### Dependency

```mermaid
classDiagram

class Client {
	+do(supplier: Supplier): Void
}

class Supplier {
	+execute(): Void
}

Client ..> Supplier: Depend
```
