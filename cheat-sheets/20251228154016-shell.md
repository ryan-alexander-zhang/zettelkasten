## Control Operators

| Operator      | Descripion                    | Demo                   |
| ------------- | ----------------------------- | ---------------------- |
| <kbd>\|</kbd> | Pipeline                      |                        |
| <kbd>;</kbd>  | Command separator by sequence | `echo a; echo b`       |
| <kbd>&</kbd>  | Backgroud                     | `sleep 10 & echo done` |

## Special Parameters

> [!NOTE] Use `man bash` and query ` Special Parameters ` in the doc.

| Param | Description                          |
| ----- | ------------------------------------ |
| `$?`  | The return code of previous command. |
