Agent skills are a set of instructions and resources. It's a packaged workflow. The agent will utilize these skills to complete tasks on demand, as described in the skill's description.

Familiar with the prompt files of the GitHub Copilot. But there are differences. Agent skills can be used automatically by the Agent.

The Agent Skills are located in the orchestration layer compared to the [[20260111114605-mcp|MCP]] in the integration layer.
