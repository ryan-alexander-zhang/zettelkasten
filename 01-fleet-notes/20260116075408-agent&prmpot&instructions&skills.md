* Agent: 自动化执行任务，调用工具访问外部，拥有记忆、可处理上下文的全自动助手
* Prompt：模块化的功能，可复用的提示词
* Instructions：用于代码生成的约束
* Skills：给人看的话跟 Prompt 文件一模一样，没啥两样，但 Skills 是 Agent 可以看得到的存在，可以让 Agent 进行选择，Prompt 可以封装成 Command 人去调用
