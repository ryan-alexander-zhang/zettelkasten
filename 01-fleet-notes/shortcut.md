## **IDEA**

| Key         | Description                             |
| ----------- | --------------------------------------- |
| `⌘ + K`     | Git Commit                              |
| `⌘` + 1     | Show Project In Tool Window             |
| `⌥ + F1`    | Select in. For example, file structure. |
| `⌘ + E`     | Recent Files.                           |
| `⌘ + ⌥ + M` | Extract the selected code as a method.  |
| `⌃ + ⌥ + O` | Optimize imports.                       |
| `⌘ + D`     | Show Git difference.                    |
| `⌘ + W`     | Close the tab.                          |
| `⌘ + ⇧ + U` | Upper case.                             |
| `⌘ + ⬅️`    | Make cursor to line head.               |
