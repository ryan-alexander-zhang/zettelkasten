> [!TIP] There are two versions of Bloom's Taxonomy; the second version is better suited for writing goals.

## Remember

Knew it now.
* How
* List
* Name
* Show
* 5W2H
## Understand

Can explain and know why.
* Explain
* Classify
## Apply

Use the knowledge to solve the problem. 
* Apply
* Build
* Choose
* Develop
## Analyze

Can find the evidence to support and find the root cause of the issue.
* Analyze
* Calculate
* Breakdown
* Diagram

> [!EXAMPLE] Can find a bug.

## Evaluate

Make judgments or do tradeoffs.
* Agree
* Choose
* Compare
* Decide
* Measure

## Create

Create a new one or enhance the existing components.
* Build
* Adapt
* Change
* Create
* Design
* Develop
* Improve
* Enhance
