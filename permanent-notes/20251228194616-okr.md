> [! INFO] OKR = Objective + Key Results

OKR is a framework for goal-setting. You can use the [[20251228184806-smart|SMART]] framework to define the objective and key results. Breaking the objective down in a MECE way if the objective is large. Can mark the [[20251228193159-bloom's-taxonomy|Bloom's Taxonomy]] cognitive complexity for the result.


> [!EXAMPLE]
> - KR1（Understand）：写 1 篇“RAG 端到端流程”说明文 + Mindmap，包含数据流与组件职责
>         
> - KR2（Apply）：完成 1 个最小 RAG Demo，可本地一键运行
>         
> - KR3（Analyze/Evaluate）：做 1 次评估实验并写 tradeoff
>         
> - KR4（Create）：把 Demo 产品化一个功能点（缓存/权限/观测任一）
